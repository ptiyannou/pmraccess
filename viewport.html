Politique de confidentialité – PMR Access

Dernière mise à jour : Mai 2026

1. Présentation

PMR Access est une application destinée à aider les utilisateurs à localiser des places de stationnement accessibles aux personnes à mobilité réduite (PMR).

2. Données collectées

L’application peut utiliser la localisation de l’utilisateur afin d’afficher les places PMR situées à proximité.

Aucune donnée personnelle nominative n’est collectée, stockée ou revendue.

3. Utilisation des données

Les données de localisation sont utilisées uniquement pour :

  • Afficher les places PMR proches de l’utilisateur
  • Améliorer l’expérience de navigation
  • Calculer les distances et itinéraires

4. Partage des données

PMR Access ne partage aucune donnée personnelle avec des tiers.

5. Services tiers

L’application peut ouvrir des applications externes telles que :

  • Apple Plans
  • Google Maps
  • Waze
  • OpenStreetMap

Ces services disposent de leurs propres politiques de confidentialité.

6. Conservation des données

Aucune donnée utilisateur n’est conservée sur des serveurs distants.

7. Contact

Pour toute question :

📧 contact@pmr-access.fr

Privacy Policy – PMR Access

Last updated: May 2026

1. Overview

PMR Access is an application designed to help users find accessible parking spaces for people with reduced mobility.

2. Data Collection

The application may use the user's location to display nearby accessible parking spaces.

No personally identifiable information is collected, stored, or sold.

3. Use of Data

Location data is used only to:

  • Display nearby accessible parking spaces
  • Improve navigation experience
  • Calculate distances and routes

4. Data Sharing

PMR Access does not share personal data with third parties.

5. Third-Party Services

The application may open external services such as:

  • Apple Maps
  • Google Maps
  • Waze
  • OpenStreetMap

These services have their own privacy policies.

6. Data Retention

No user data is stored on remote servers.

7. Contact

For any questions:

📧 contact@pmr-access.fr
